<h2>Installation</h2>

* The user can clone the repo from GitHub. 
* In your CLI, run `npm i` to install dev dependencies.
* Run `npm start` to start up the server.

<h2>Technologies Used</h2>

* Built with:
    - React
    - Material UI
    - JavaScript
    - HTML/JSX
